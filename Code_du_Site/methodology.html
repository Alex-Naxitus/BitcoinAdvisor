    • Introduction

    • The best predictions whether political or business are currently being made by computers, through Machine Learning Algorithms.
    • Old and simple methods like Moving Averages (SMA), Relative Strength Index (RSI), and Moving Average Convergence Divergence (MACD) aren't used anymore because they're not enough accurate.
    • Today, powerful algorithms train themselves to predict data by using extremely complex mathematical formulas to compute the relationships between thousands of datasets.

    The standard predictions

    • There are already several Machine Learning models (like Gaussian Process Regressor) that are good at predicting low-volatibility stocks.
    • Since the price of Bitcoin is volatile, we cannot just use the existing models.
    • As you can see, we tried using standard machine learning to predict the price, but these predictions are not volatile enough.

    How are the predictions made?

    1. Unlike standard models, we will not use time to predict prices ; prices are determined by Human Action.
    2. We used historical Macroeconomic and Financial features like interest rates, the values of stocks, bonds, forex ...etc.
    3. We selected features that have the highest correlation with the price of Bitcoin.
    4. We created a Machine Learning algorithm based on those selected features.
    5. We therefore needed to combine the old and the new methods to predict the price of Bitcoin.
